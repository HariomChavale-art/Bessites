# Webdock

Professional discovery engine and creator suite.

## Git Configuration

To verify your current remote:
```bash
npm run check-git
```

To set the remote URL manually:
```bash
git remote set-url origin https://github.com/HariomChavale-art/Webdock.git
```

## Quick Commands

- `npm run dev`: Start development server
- `npm run sync`: Commit and push changes to GitHub (Webdock repository)
- `npm run pull-git`: Pull latest changes from GitHub with rebase
- `npm run check-git`: Verify the remote Git URL
- `npm run auth-git`: Setup credential storage for GitHub
- `npm run fix-git`: Repair branch tracking and origin setup

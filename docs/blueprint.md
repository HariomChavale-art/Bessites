# **App Name**: NetFlow

## Core Features:

- Sponsored Horizon Carousel: An infinite horizontally moving banner at the top of the page featuring high-priority featured websites and sponsors.
- Personalized Masonry Grid: An addictive, infinite-scroll Pinterest-inspired layout that organizes web application cards based on visual hierarchy.
- AI Content Discovery Tool: An AI tool that reasons about a user's browsing patterns and favorite categories to dynamically suggest niche websites in the primary feed.
- Play Store Info Panels: Detailed modal or page view for each website including metadata, interactive screenshots, and technical specs based on Play Store layouts.
- Review & Rating System: A structured section for user feedback and numerical ratings to determine the credibility and popularity of listed websites.
- Intelligent Category Tagging: Automated categorization system using generative AI to analyze submitted website URLs and assign them into highly relevant Pinterest-style interests.

## Style Guidelines:

- The scheme follows a 'Midnight Pulse' theme: Primary color is Electric Indigo (#7B33FF), background is a deeply desaturated Slate (#18151F), and the accent is a luminous Periwinkle (#85A3FF).
- A font pairing using 'Poppins' for high-impact headlines and UI elements to capture an avant-garde feel, and 'Inter' for body text and long descriptions to ensure objective clarity.
- Hybrid masonry architecture: Dynamic-height grid columns that prioritize visual content, transitioning to a structured 3-column metadata layout for detail pages.
- Liquid scrolling effects and infinite marquee transformations for the hero section to maximize user retention.
- Line-based minimalist icons with slight outer-glows that reflect the Indigo primary color to signify high-tech interactivity.

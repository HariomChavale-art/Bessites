# Webdock

Professional discovery engine and creator suite.

## Git Configuration

To set the remote URL manually:
```bash
git remote set-url origin https://github.com/HariomChavale-art/Webdock.git
```

## Quick Commands

- `npm run dev`: Start development server
- `npm run sync`: Commit and push changes to GitHub
- `npm run auth-git`: Setup credential storage
